# Blood Donation Management System

This is a web-based application designed to manage blood donation activities efficiently.

## Features
- Donor registration
- Blood group search
- Request blood functionality
- Simple and user-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript / Python (तुझं जे असेल ते)

## Purpose
This project was built to help manage blood donation data and make it easier to connect donors and recipients.

## Status
This project is actively maintained and open for improvements.
